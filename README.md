- `data/secondhand.json` - Second-hand market analysis

## Methodology

1. Searched 12+ Australian retailers
2. Checked OzBargain for current deals
3. Analysed refurbished/second-hand markets
4. Verified Perth delivery options
5. Compiled interactive comparison dashboard
